## Usage
### Tmux
Kill pane                : C-x 0
Split window vertically  : C-x 2
------------ horizontally: C-x 3
Next pane                : C-x o
Grow pane                : C-x M-up/down/left/right
Show all keys            : C-x ?
Rotate panes             : C-x C-o
